Stable Releases

LibVMI releases are shipped when a new set of features has reached sufficient stability. Starting with the 0.12 release, we will begin backporting bug fixes and providing maintenance releases, as needed.

Stable releases can be verfied using GnuPG and key 3045D1E5 from the MIT PGP public key server.
